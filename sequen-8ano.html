Sequências Matemáticas — 8º Ano

Guia de estudos sobre Padrões, P.A. e P.G.

O que são Sequências?

Uma sequência numérica é uma lista de números escritos em uma ordem específica. Cada número da lista é chamado de termo. As sequências podem ser finitas (com um fim) ou infinitas (sem fim).

  • Exemplo Finito: (2, 4, 6, 8, 10) - Os números pares positivos até 10.
  • Exemplo Infinito: (3, 6, 9, 12, ...) - Os múltiplos de 3.

Muitas sequências seguem um padrão ou uma regra, que chamamos de lei de formação. Essa lei nos permite encontrar qualquer termo da sequência.

Progressão Aritmética (PA)

Uma Progressão Aritmética (PA) é uma sequência em que cada termo, a partir do segundo, é igual ao anterior somado a um valor constante, chamado de razão (r).

Exemplo: (5, 8, 11, 14, ...) é uma PA.
Para achar a razão, subtraia um termo pelo seu anterior: 8 - 5 = 3. A razão (r) é 3.

Fórmula do Termo Geral da PA

Para encontrar qualquer termo (aₙ) de uma PA sem precisar escrever todos os anteriores, usamos a fórmula:

aₙ = a₁ + (n - 1) ⋅ r

Onde: aₙ é o termo que queremos encontrar, a₁ é o primeiro termo, n é a posição do termo, e r é a razão.

Progressão Geométrica (PG)

Uma Progressão Geométrica (PG) é uma sequência em que cada termo, a partir do segundo, é igual ao anterior multiplicado por um valor constante, chamado de razão (q).

Exemplo: (2, 6, 18, 54, ...) é uma PG.
Para achar a razão, divida um termo pelo seu anterior: 6 / 2 = 3. A razão (q) é 3.

Fórmula do Termo Geral da PG

Para encontrar qualquer termo (aₙ) de uma PG, usamos a fórmula:

aₙ = a₁ ⋅ qⁿ⁻¹

Onde: aₙ é o termo que queremos encontrar, a₁ é o primeiro termo, q é a razão, e n é a posição do termo.
